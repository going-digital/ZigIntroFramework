# Zig 4k intro framework

A pure Zig (+GLSL) framework for sizecoding intros, targetting PC, 32 bit.

## Project status
Visuals now working, but there is only a token audio synth.

## Build instructions

Use zig version 0.13.0. `zig build` to compile.
